# My Work Showcase

* Word scramble｜可以學習英語單詞的單詞拼接遊戲，有中文翻譯功能所以也可以用作記單詞軟件。（使用 google ml kit 中的translation 功能）
  -
* Time unit converter｜時間單位換算小工具，將來會加入更多單位換算功能。
  -
* Estimate ur sleeping hours ｜動用 apple machine learning kit 來幫用戶算所需睡眠時間的小工具。
  -
* Guess the flags ｜猜各國國旗的遊戲。
  -
* Rock Paper Scissor Game ｜無聊嗎？ 和電腦玩猜拳吧 🙄️。
  -
* Fun Multiplication ｜ 適合小朋友練習乘法口訣的小遊戲。
  -
* Bill Splitting App ｜和朋友AA的好工具。
  -
* Food Order app | 订餐app，可以将数据传输到网上。
  -
* Coming soon ~
